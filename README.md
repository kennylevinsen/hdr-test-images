# hdr-test-images

A selection of generated HDR test images in variors formats and color parameters. These test images are meant to render identically to the SDR reference JPEG, and do not exercise the extended range.

- Image formats: PNG, AVIF, HEIF and JPEG-XL
- Transfer functions: HLG (1000-nit target), PQ, sRGB + gain map
- Primaries: BT.2020, DCI-P3

You can use [tev](https://github.com/Tom94/tev) to view the examples as they were intended to be rendered.
